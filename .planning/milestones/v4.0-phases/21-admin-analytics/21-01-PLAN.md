**New endpoints:**

**1. GET /admin/overview (mounted as /api/analytics/admin/overview)**
Returns site-wide aggregate counts.

SQL queries:
```sql
SELECT COUNT(*) AS total FROM posts WHERE status = 'published';
SELECT COUNT(*) AS total FROM post_likes;
SELECT COUNT(*) AS total FROM post_comments;
SELECT COUNT(*) AS total FROM users WHERE role IN ('contributor', 'admin') AND is_active = 1;
SELECT COUNT(*) AS total FROM users WHERE role = 'reader' AND is_active = 1;
```

Run all five queries in parallel with `Promise.all` for performance.

Return shape:
```json
{
  "totalPosts": number,
  "totalLikes": number,
  "totalComments": number,
  "totalContributors": number,
  "totalReaders": number
}
```

**2. GET /admin/contributors (mounted as /api/analytics/admin/contributors)**
Returns contributors ranked by post count + total engagement. Limit to top 20.

SQL query:
```sql
SELECT u.id, u.username, u.display_name,
       COUNT(DISTINCT p.id) AS post_count,
       COALESCE(SUM(lc.like_count), 0) AS total_likes,
       COALESCE(SUM(cc.comment_count), 0) AS total_comments
FROM users u
LEFT JOIN posts p ON p.author_id = u.id AND p.status = 'published'
LEFT JOIN (SELECT post_id, COUNT(*) AS like_count FROM post_likes GROUP BY post_id) lc ON lc.post_id = p.id
LEFT JOIN (SELECT post_id, COUNT(*) AS comment_count FROM post_comments GROUP BY post_id) cc ON cc.post_id = p.id
WHERE u.role IN ('contributor', 'admin') AND u.is_active = 1
GROUP BY u.id
ORDER BY post_count DESC, (total_likes + total_comments) DESC, u.display_name ASC
LIMIT 20
```

Return shape:
```json
{
  "contributors": [{
    "username": string,
    "displayName": string,
    "postCount": number,
    "totalLikes": number,
    "totalComments": number,
    "totalEngagement": number
  }]
}
```

Compute `totalEngagement` as `totalLikes + totalComments` in JS when mapping the rows.

**3. GET /admin/artists (mounted as /api/analytics/admin/artists)**
Returns top artists across ALL contributors, ranked by how many posts mention them. Limit to 20.

SQL query:
```sql
SELECT pa.artist_name, pa.artist_image, COUNT(DISTINCT pa.post_id) AS post_count
FROM post_artists pa
JOIN posts p ON p.id = pa.post_id
WHERE p.status = 'published'
GROUP BY pa.artist_name
ORDER BY post_count DESC, pa.artist_name ASC
LIMIT 20
```

Return shape:
```json
{
  "artists": [{ "name": string, "image": string, "postCount": number }]
}
```

Wrap each endpoint handler in try/catch, log errors with `console.error`, return 500 with `{ error: "..." }`.
