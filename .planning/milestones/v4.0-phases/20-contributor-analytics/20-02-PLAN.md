  <action>
Create client/src/pages/Stats.jsx — a contributor stats dashboard. Use the same layout patterns as other pages (max-w-3xl mx-auto px-4 py-8).

**Structure:**

1. **State:**
   - `posts` (array), `totalLikes`, `totalComments` — from myStats
   - `artists` (array) — from myArtists
   - `activity` ({ totalPosts, postsThisMonth, currentStreak }) — from myActivity
   - `sort` — 'likes' | 'comments' | 'recent', default 'likes'
   - `loading` — true initially
   - `error` — null or error string

2. **Data fetching:**
   Use two useEffect hooks:
   - First effect (runs once on mount): fetch `analytics.myArtists()` and `analytics.myActivity()` in parallel with Promise.all. Set artists and activity state.
   - Second effect (runs when `sort` changes): fetch `analytics.myStats(sort)`. Set posts, totalLikes, totalComments. Set loading=false after both effects complete on initial load.

   Handle errors with try/catch, set error state, set loading=false.

3. **Layout — four sections:**

   **a. Activity summary cards (top row)**
   Three stat cards in a horizontal row (grid grid-cols-3 gap-4):
   - "Total Posts" — activity.totalPosts (large number, label below)
   - "This Month" — activity.postsThisMonth
   - "Streak" — activity.currentStreak, append "days" as suffix

   Card style: bg-white dark:bg-gray-900 border border-gray-200 dark:border-gray-800 rounded-lg p-4 text-center. Number: text-3xl font-bold text-gray-900 dark:text-gray-100. Label: text-sm text-gray-500 dark:text-gray-400 mt-1.

   **b. Engagement totals (below activity cards)**
   Two inline stats: "Total likes: {totalLikes}" and "Total comments: {totalComments}" — styled as text-sm text-gray-600 dark:text-gray-400, separated by a middot or pipe.

   **c. Post performance table**
   Heading: "Post Performance" (text-lg font-semibold)

   Sort controls: three buttons (Likes, Comments, Recent) styled as tab-like toggles. Active tab: text-gray-900 dark:text-gray-100 border-b-2 border-gray-900 dark:border-gray-100. Inactive: text-gray-400 hover:text-gray-600. Buttons in a row with gap-4, mb-4.

   Post list: for each post, a row showing:
   - Left side: embed title (or truncated body if no embed title) as a Link to `/posts/${post.slug}`, text-sm font-medium text-gray-900 dark:text-gray-100 hover:underline. Below it: published date (formatted as "Mar 19, 2026" using toLocaleDateString) in text-xs text-gray-400.
   - Right side: like count with a heart icon and comment count with a comment icon, text-sm text-gray-500. Use simple inline SVG icons (small, 16x16).

   Each row: flex items-center justify-between py-3 border-b border-gray-100 dark:border-gray-800. Last row no border.

   If no posts: "No published posts yet." text-gray-400 text-sm.

   **d. Top Artists**
   Heading: "Top Artists" (text-lg font-semibold mt-8)

   List of artists, each row showing:
   - Artist image (24x24 rounded-full, or a gray placeholder circle if no image) on the left
   - Artist name as a Link to `/artist/${encodeURIComponent(artist.name)}`, text-sm text-gray-900 dark:text-gray-100 hover:underline
   - Post count on the right: text-sm text-gray-400, e.g. "3 posts"

   Each row: flex items-center gap-3 py-2.

   If no artists: "No artists tagged yet." text-gray-400 text-sm.

4. **Loading state:** Show a centered "Loading..." text-gray-400 while loading is true.

5. **Error state:** Show error message in text-red-600 text-sm.

6. **Page title:** `<h1 className="text-2xl font-bold text-gray-900 dark:text-gray-100 mb-6">Stats</h1>`

Import analytics from '../api/client.js'. Import useState, useEffect from react. Import Link from react-router-dom.
