Create server/routes/analytics.js with three endpoints, all gated by authenticateToken + requireContributor:

**1. GET / (mounted as /api/analytics/me)**
Returns per-post stats for the authenticated contributor. Accept query param `sort` with values `likes` (default), `comments`, or `recent`.

SQL query — join posts with aggregated likes and comments:
```sql
SELECT p.id, p.slug, p.body, p.created_at, p.published_at,
       pe.title AS embed_title, pe.thumbnail_url, pe.provider,
       COALESCE(lc.like_count, 0) AS like_count,
       COALESCE(cc.comment_count, 0) AS comment_count
FROM posts p
LEFT JOIN post_embeds pe ON pe.post_id = p.id
LEFT JOIN (SELECT post_id, COUNT(*) AS like_count FROM post_likes GROUP BY post_id) lc ON lc.post_id = p.id
LEFT JOIN (SELECT post_id, COUNT(*) AS comment_count FROM post_comments GROUP BY post_id) cc ON cc.post_id = p.id
WHERE p.author_id = ? AND p.status = 'published'
ORDER BY {sort_column} DESC
```

Where sort_column is:
- `likes` → `like_count DESC, p.published_at DESC`
- `comments` → `comment_count DESC, p.published_at DESC`
- `recent` → `p.published_at DESC`

Whitelist the sort param — default to `likes` if invalid value provided. Do NOT interpolate sort directly into SQL; use an if/else or object lookup to pick the ORDER BY clause string (safe from injection since it's from a fixed set).

Also compute totals in the response: `totalLikes` (sum of all like_count), `totalComments` (sum of all comment_count).

Return shape:
```json
{
  "posts": [{ "slug", "body" (first 120 chars), "embedTitle", "thumbnailUrl", "provider", "publishedAt", "likeCount", "commentCount" }],
  "totalLikes": number,
  "totalComments": number
}
```

Truncate body to first 120 characters + "..." for the stats view (no need for full post body). Use camelCase keys in the response (convert from snake_case).

**2. GET /artists (mounted as /api/analytics/me/artists)**
Returns the contributor's most-written-about artists with post counts.

SQL:
```sql
SELECT pa.artist_name, pa.artist_image, COUNT(DISTINCT pa.post_id) AS post_count
FROM post_artists pa
JOIN posts p ON p.id = pa.post_id
WHERE p.author_id = ? AND p.status = 'published'
GROUP BY pa.artist_name
ORDER BY post_count DESC, pa.artist_name ASC
LIMIT 20
```

Return shape:
```json
{
  "artists": [{ "name", "image", "postCount" }]
}
```

**3. GET /activity (mounted as /api/analytics/me/activity)**
Returns activity stats: total published posts, posts this month, and current posting streak (consecutive days with at least one published post, counting backward from today).

SQL for total and this month:
```sql
SELECT COUNT(*) AS total_posts FROM posts WHERE author_id = ? AND status = 'published';
SELECT COUNT(*) AS posts_this_month FROM posts WHERE author_id = ? AND status = 'published' AND published_at >= date('now', 'start of month');
```

For posting streak — fetch all distinct publish dates (DATE(published_at)) for the contributor, ordered descending. Then iterate from today backward: if today has a post, streak starts at 1 and increments for each consecutive prior day with a post. If today has no post but yesterday does, start from yesterday. If neither today nor yesterday has a post, streak is 0.

SQL for streak dates:
```sql
SELECT DISTINCT DATE(published_at) AS pub_date
FROM posts
WHERE author_id = ? AND status = 'published'
ORDER BY pub_date DESC
LIMIT 365
```

Then compute streak in JS:
```js
function computeStreak(dates) {
  if (!dates.length) return 0;
  const today = new Date();
  today.setHours(0, 0, 0, 0);
  const todayStr = today.toISOString().slice(0, 10);
  const yesterdayStr = new Date(today - 86400000).toISOString().slice(0, 10);

  let streak = 0;
  let checkDate = todayStr;

  // If no post today, check if yesterday has one (grace period)
  if (dates[0] !== todayStr) {
    if (dates[0] !== yesterdayStr) return 0;
    checkDate = yesterdayStr;
  }

  for (const d of dates) {
    if (d === checkDate) {
      streak++;
      const prev = new Date(checkDate);
      prev.setDate(prev.getDate() - 1);
      checkDate = prev.toISOString().slice(0, 10);
    } else if (d < checkDate) {
      break;
    }
  }
  return streak;
}
```

Return shape:
```json
{
  "totalPosts": number,
  "postsThisMonth": number,
  "currentStreak": number
}
```
