**client/src/components/LikeButton.jsx:**

Create new component with these specs:

Props: `{ postSlug, initialCount, initialLiked }`

1. Import `useState, useRef` from React, `useNavigate` from react-router-dom, `useAuth` from AuthContext, `posts` from api/client.
2. State: `liked` (init from `initialLiked`), `count` (init from `initialCount`). Ref: `togglingRef` (useRef, init false) for debounce.
3. `handleToggle` async function:
   - If `!user`: navigate to `/join` and return.
   - If `togglingRef.current` is true: return (prevents rapid double-tap).
   - Set `togglingRef.current = true`.
   - Optimistic update: save previous state, set `liked` to `!liked`, set `count` to `liked ? count - 1 : count + 1`.
   - Try: `const res = await posts.like(postSlug)` -- then set `liked` and `count` from `res.data`.
   - Catch: roll back to saved previous state.
   - Finally: set `togglingRef.current = false`.
4. Render a `<button>` with `onClick={handleToggle}`:
   - `className`: flex items-center gap-1 text-sm transition group. Use `aria-label` for accessibility: "Like" or "Unlike".
   - Inside: an inline SVG (24x24 viewBox, w-5 h-5 class).
     - When `liked`: filled heart path (solid red, `fill="currentColor"` with `text-red-500`).
     - When not liked: outline heart path (`stroke="currentColor"` with `text-gray-400 dark:text-gray-500 group-hover:text-red-400`).
   - After SVG: `<span>` with count. Show count only when `count > 0`. Style: `text-gray-500 dark:text-gray-400`.

Heart SVG paths (from research):
- Outline: `<path strokeLinecap="round" strokeLinejoin="round" strokeWidth={1.5} d="M21 8.25c0-2.485-2.099-4.5-4.688-4.5-1.935 0-3.597 1.126-4.312 2.733-.715-1.607-2.377-2.733-4.313-2.733C5.1 3.75 3 5.765 3 8.25c0 7.22 9 12 9 12s9-4.78 9-12z" />`
- Filled: `<path d="M11.645 20.91l-.007-.003-.022-.012a15.247 15.247 0 01-.383-.218 25.18 25.18 0 01-4.244-3.17C4.688 15.36 2.25 12.174 2.25 8.25 2.25 5.322 4.714 3 7.688 3A5.5 5.5 0 0112 5.052 5.5 5.5 0 0116.313 3c2.973 0 5.437 2.322 5.437 5.25 0 3.925-2.438 7.111-4.739 9.256a25.175 25.175 0 01-4.244 3.17 15.247 15.247 0 01-.383.219l-.022.012-.007.004-.003.001a.752.752 0 01-.704 0l-.003-.001z" />`

Both SVGs use `viewBox="0 0 24 24"` and `xmlns="http://www.w3.org/2000/svg"`. The outline version uses `fill="none"` with `stroke="currentColor"`. The filled version uses `fill="currentColor"` with no stroke.

The component manages its own count and liked state independently from the parent post data (per research Pitfall 5). No state lifting needed.
