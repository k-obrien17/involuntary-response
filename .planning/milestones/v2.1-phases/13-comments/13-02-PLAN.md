**client/src/components/CommentSection.jsx:**
Create a new component following the LikeButton self-managed state pattern.

Props: `{ postSlug, initialComments, postAuthorId }`

State:
- `comments` (array, initialized from initialComments || [])
- `body` (string, the compose textarea value)
- `submitting` (boolean, prevents double-submit)

Imports: `useState` from react, `useAuth` from `../context/AuthContext`, `posts` from `../api/client`, `Avatar` from `./Avatar`, `relativeTime` from `../utils/formatDate`.

**Structure:**
1. **Comment count header:** Display "N comments" (or "No comments yet" when empty). Use `comments.length`.

2. **Comment list:** Map over `comments` array. Each comment renders:
   - `Avatar` component with `emailHash={comment.author.emailHash}` `displayName={comment.author.displayName}` `size={24}`
   - Author display name as a text span (not a link -- readers don't have profile pages per Out of Scope)
   - Relative timestamp via `relativeTime(comment.createdAt)`
   - Comment body as a `<p>` tag
   - Delete button (trash icon or "Delete" text) -- only shown when `comment.canDelete === true`
   - Delete handler: call `posts.deleteComment(postSlug, comment.id)`, on success filter the comment out of local state (optimistic removal: remove immediately, restore array on error per research Pitfall 5)

3. **Compose form** (only rendered when `user` is not null):
   - `<textarea>` with value={body}, onChange updates body, maxLength={500}, placeholder="Leave a comment..."
   - Character count indicator showing `{body.length}/500` (only visible when body.length > 0)
   - Submit button: "Post" text, disabled when `submitting || !body.trim()`
   - onSubmit: preventDefault, call `posts.addComment(postSlug, body.trim())`, on success append `res.data` to comments array and clear body. Set submitting true/false around the call.

4. **Logged-out message:** When `user` is null, show a subtle prompt below the comment list: "Log in to leave a comment" with "Log in" as a Link to `/join`.

**Tailwind styling (consistent with existing components):**
- Section wrapper: `mt-10 border-t border-gray-200 dark:border-gray-700 pt-8`
- Comment count header: `text-sm font-medium text-gray-700 dark:text-gray-300 mb-6`
- Each comment: `mb-6` wrapper, flex layout for avatar + meta, `text-sm` for meta text
- Author name: `font-medium text-gray-700 dark:text-gray-300`
- Timestamp: `text-gray-400 dark:text-gray-500`
- Comment body: `mt-1 text-gray-600 dark:text-gray-400`
- Delete button: `text-xs text-gray-400 dark:text-gray-500 hover:text-red-500 dark:hover:text-red-400 transition` -- use text "Delete", not an icon (simpler, matches existing patterns)
- Textarea: `w-full rounded-md border border-gray-300 dark:border-gray-600 bg-white dark:bg-gray-800 px-3 py-2 text-sm text-gray-900 dark:text-gray-100 placeholder-gray-400 dark:placeholder-gray-500 focus:ring-1 focus:ring-gray-400 dark:focus:ring-gray-500 focus:outline-none resize-none`
- Submit button: `px-4 py-1.5 text-sm font-medium rounded-md bg-gray-900 dark:bg-gray-100 text-white dark:text-gray-900 hover:bg-gray-700 dark:hover:bg-gray-300 disabled:opacity-50 transition`
- Log in prompt: `text-sm text-gray-400 dark:text-gray-500` with the Link styled as `underline hover:text-gray-600 dark:hover:text-gray-300`
